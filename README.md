# 🌍 FlagMaster - Guess the Country by Flag

FlagMaster is a fun and interactive React-based quiz game where users test their knowledge by identifying countries based on their flags.

---

## 🚀 Features

* 🌎 Random country flag generation
* 🎯 Multiple choice answers
* 📈 Score tracking system
* 🔄 New question on every attempt
* ⚡ Fast and responsive UI
* 🌐 Real-time data using REST Countries API

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🌍 REST Countries API


---

## 📡 API Used

This project uses the REST Countries API to fetch country names and flag images.

Endpoint:
https://restcountries.com/v3.1/all

---

## 📸 How It Works

1. A random country flag is displayed
2. Four options are shown
3. User selects the correct country
4. Score updates instantly
5. Next question loads automatically
